Notice d'Utilisation - Calculateur d'Insuline

Étape 1 : Remplir les Informations Personnelles

Sur la page principale, vous verrez un formulaire vous demandant les informations suivantes :

  • Poids (en kilogrammes) : Entrez votre poids actuel.
  • Glycémie Actuelle (en mg/dL) : Entrez votre glycémie mesurée récemment.
  • Glycémie Cible (en mg/dL) : Entrez la valeur cible de glycémie à atteindre.
  • Facteur de Sensibilité à l’Insuline : Indiquez votre facteur de sensibilité (souvent fourni par votre médecin). Cela représente combien de mg/dL votre glycémie baisse par unité d’insuline.

Étape 2 : Entrer les Aliments Consommés

Sous les informations personnelles, vous pouvez ajouter les aliments que vous avez consommés et leur quantité :

  • Aliments : Tapez le nom de chaque aliment que vous avez consommé (par exemple, “pomme”, “riz”, “poulet”).
  • Quantité : Indiquez la quantité consommée en grammes.

Exemple : Si vous avez mangé 100g de riz, entrez “riz” dans la colonne des aliments et “100” dans la colonne des quantités.

Étape 3 : Calculer la Dose d’Insuline

Une fois que vous avez rempli toutes les informations nécessaires (poids, glycémie, aliments), cliquez sur le bouton Calculer.

Étape 4 : Résultats

Le calculateur vous fournira trois informations importantes :

  1. Dose d’Insuline Recommandée : La quantité d’insuline (en unités) que vous devez administrer pour compenser les glucides consommés et corriger votre glycémie.
  2. Ratio Insuline/Glucides : Le nombre de grammes de glucides que 1 unité d’insuline peut couvrir.
  3. Total des Glucides Consommés : La quantité totale de glucides (en grammes) que vous avez consommée, basée sur les aliments saisis.

Clause de Non-Responsabilité

L’utilisation de ce Calculateur d’Insuline est à vos propres risques. Bien que l’application soit conçue pour fournir des estimations basées sur les données que vous entrez, elle ne remplace en aucun cas l’avis médical d’un professionnel de la santé.

Avertissement : Les résultats fournis par cette application sont basés sur les informations que vous avez saisies. Le développeur de l’application ne peut être tenu responsable des conséquences résultant de l’utilisation de ces données ou de la non-conformité à vos recommandations médicales. Veuillez consulter votre médecin ou un professionnel de santé qualifié avant de prendre des décisions concernant votre traitement.
